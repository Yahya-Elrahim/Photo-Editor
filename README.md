# 📷 Photo Editor

A modern, fast, and feature-rich Photo Editor that helps users enhance, retouch, and transform images with professional-quality editing tools. Whether you need quick adjustments or advanced editing, this application provides an intuitive experience for everyone.

## ✨ Features

### 🖼️ Image Editing
- Crop, rotate, flip, and resize images
- Free rotation and straightening
- Undo & redo support
- Auto enhancement

### 🎨 Adjustments
- Brightness
- Contrast
- Saturation
- Vibrance
- Exposure
- Highlights & Shadows
- Temperature & Tint
- Sharpness
- Blur
- Vignette

### 🌈 Filters & Effects
- Professional photo filters
- Vintage
- Black & White
- Cinematic
- HDR
- Retro
- Warm & Cool tones
- Artistic effects
- Adjustable filter intensity

### ✍️ Drawing & Text
- Draw with customizable brushes
- Add text with multiple fonts
- Colors, gradients, shadows, and outlines
- Shapes (rectangle, circle, arrow, line)
- Stickers and emojis

### 🖌️ Advanced Editing
- Layer support
- Blend modes
- Opacity controls
- Clone tool
- Healing tool
- Perspective correction
- Masking
- Selective editing

### 🖼️ Background Tools
- Remove background
- Replace background
- Blur background
- Transparent background support
- Solid, gradient, and pattern backgrounds

### 🎭 AI Features
- AI image enhancement
- AI object removal
- AI background removal
- AI portrait retouching
- AI skin smoothing
- AI face enhancement
- AI image upscaling
- AI color correction

### 📐 Collages & Frames
- Photo collage layouts
- Custom grids
- Decorative frames
- Borders
- Rounded corners

### 📤 Export
- JPG
- PNG
- WEBP
- PDF
- High-quality export
- Transparent PNG support
- Multiple resolutions

### ☁️ Sharing
- Save locally
- Share directly to social media
- Copy to clipboard
- Cloud backup support

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/photo-editor.git
```

### Navigate to the project

```bash
cd photo-editor
```

### Install dependencies

Follow the setup instructions for your development environment.

### Run the application

```bash
# Example
npm install
npm run dev
```

or

```bash
./gradlew run
```

depending on your project.
